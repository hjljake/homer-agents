# Funnel Continuity Audit

A landing page doesn't exist in isolation. It's one stage in a journey that starts when someone first becomes aware of the brand and ends when they become a paying customer (or fail to). Optimizing the page without understanding the stages before and after it is like editing the middle chapter of a book without reading the beginning or end.

Most conversion problems aren't page problems — they're journey problems. The ad promises one thing, the page delivers another, and the post-click experience delivers a third. Each disconnection bleeds conversions.

This skill provides the framework for auditing the entire funnel as a single connected system.

---

## The Five Funnel Stages

Every customer acquisition funnel passes through these stages, regardless of business model or traffic source. The specifics differ, but the stages are universal.

```

│                                                         │
│   STAGE 1: AWARENESS                                    │

│                                                         │

│                                                         │

│                                                         │

│   invest attention. The click is a micro-contract:      │

│                                                         │
```

## The Continuity Principles

### 1. Message Continuity
The core promise must thread through every stage without mutation. If the ad says "Save 3 hours per week on reporting," the landing page headline should echo that promise, the CTA should reference it, and the thank-you page should reinforce it.

**How to audit:** Extract the core promise from each stage. Line them up. Do they say the same thing in different words, or do they say different things?

**Common failures:**
- Ad promises a discount, page leads with brand story
- Email subject says "exclusive offer," page shows the standard pricing page
- Social post highlights Feature A, landing page leads with Feature B
- Search ad matches the query, landing page is generic

### 2. Emotional Arc
The visitor's emotional state changes at each stage. The funnel must match and advance that state, not fight it.

```
Stage 1 (Awareness):    Curiosity / Recognition of problem
Stage 2 (Click):        Interest / Hope / "This might be the answer"
Stage 3 (Landing Page): Desire / Trust / "I believe this works"
Stage 4 (Conversion):   Confidence / "This is easy and safe"
Stage 5 (Post-Convert): Relief / Excitement / "I made the right choice"
```

**How to audit:** At each stage, identify the dominant emotion. Does the next stage build on it or reset it?

**Common failures:**
- Ad creates urgency, landing page starts with "About Us" (resets emotion)
- Landing page builds desire, checkout flow creates anxiety (friction kills confidence)
- Great onboarding email, then radio silence for 2 weeks (excitement decays)

### 3. Promise Escalation
Each stage should ask for slightly more commitment while delivering slightly more value. Never ask for a big leap without earning it.

```
Stage 1: Promise value → ask for attention (low commitment)
Stage 2: Deliver on the attention promise → ask for a click (medium commitment)
Stage 3: Deliver on the click promise → ask for information or money (high commitment)
Stage 4: Make the commitment process easy → deliver immediate value signal
Stage 5: Deliver on ALL promises → ask for loyalty / referral
```

**How to audit:** At each stage, what is being asked vs what has been delivered so far? Is the ask proportional to the trust earned?

**Common failures:**
- Asking for phone number before demonstrating any value
- Asking for a purchase from a first-time visitor (cold traffic)
- Asking for a referral before the product has delivered results

### 4. Visual & Tonal Continuity
The brand should feel like the same entity across every touchpoint. Color palette, typography, tone of voice, image style, and overall "feel" should be consistent.

**How to audit:** Place screenshots of each stage side by side. Could a visitor tell they're from the same brand without reading the text?

**Common failures:**
- Ad uses playful, colorful creative; landing page is corporate and sterile
- Email is warm and personal; landing page is generic template
- Social media is edgy; checkout flow is dry and institutional

### 5. Trust Accumulation
Trust is built incrementally across stages. Each stage should add a new trust signal, not repeat the same ones.

```
Stage 1: Brand recognition / authority signal (logo, endorsement)
Stage 2: Social proof snippet ("Join 10,000+ marketers")
Stage 3: Detailed proof (testimonials, case studies, reviews)
Stage 4: Security signals (SSL, trust badges, guarantees)
Stage 5: Delivery proof ("Your account is ready" / "We received your order")
```

**How to audit:** List every trust signal at each stage. Are they accumulating? Are there gaps where trust is assumed but not earned?

## Funnel Mapping By Traffic Source

Different traffic sources create different funnel shapes. The audit must account for where the visitor enters.

### Paid Search Funnel
```
Search query (intent signal) → Ad headline + description → Landing page → Form/purchase → Thank you + email
```
**Key audit point:** Does the ad headline match the search query? Does the landing page headline match the ad? Search visitors have HIGH intent — the funnel should get out of the way and let them convert.

### Paid Social Funnel
```
Feed scroll (interrupt) → Ad creative + copy → Landing page → Form/purchase → Thank you + email
```
**Key audit point:** The ad is an INTERRUPT — the visitor wasn't looking for you. The landing page must re-establish the hook, not assume continued interest. Social funnels need more persuasion on the page.

### Email Funnel
```
Subject line (open trigger) → Email body + CTA → Landing page → Form/purchase → Thank you/confirmation
```
**Key audit point:** The subject line is a promise. The email body is the pitch. The landing page must deliver on the subject line's promise IMMEDIATELY — warm visitors who don't see what they were promised within 2 seconds will bounce.

### Organic / SEO Funnel
```
Search query → SERP snippet (title + description) → Content / landing page → Internal CTA → Form/purchase → Thank you
```
**Key audit point:** SEO funnels often have an extra stage — the visitor lands on content first, then must be guided to a conversion page. The content-to-landing-page handoff is a common leak point.

### Retargeting Funnel
```
Previous visit (memory) → Retargeting ad (reminder) → Landing page (may be same or different) → Form/purchase → Thank you
```
**Key audit point:** Retargeting visitors have PRIOR CONTEXT. The page should acknowledge this. "Welcome back" or "Still interested in X?" performs better than treating them like first-time visitors.

### Referral / Affiliate Funnel
```
Referrer context (blog post, friend, partner) → Referral link → Landing page → Form/purchase → Thank you
```
**Key audit point:** The visitor trusts the REFERRER, not you. The landing page should leverage the referrer's endorsement and match the referrer's framing of your product.

### Multi-Touch Funnel
```
Social post → Blog visit → Retargeting ad → Email signup → Nurture sequence → Sales email → Landing page → Purchase
```
**Key audit point:** Most real funnels are multi-touch. Audit the LONGEST common path, not just the last click. Identify where people drop off in the sequence.

## Stage-by-Stage Audit Criteria

### Stage 1: Awareness Audit

| Dimension | What to Evaluate |
|-----------|-----------------|
| **Channel fit** | Is the awareness channel appropriate for the ICP? (e.g., LinkedIn for B2B, Instagram for DTC) |
| **Hook quality** | Does the first impression create enough curiosity or recognition to earn continued attention? |
| **Promise clarity** | Is there a clear value proposition, even at the awareness stage? Not a full pitch — just enough to establish "this is for people like me" |
| **ICP targeting** | Does the awareness creative reach the right audience? Are targeting parameters tight enough? |
| **Volume vs quality** | Is the awareness stage optimized for reach or for qualified attention? |

### Stage 2: Click-Through Audit

| Dimension | What to Evaluate |
|-----------|-----------------|
| **Promise specificity** | Does the click trigger (ad copy, email body, social caption) make a specific promise about what the visitor will find? |
| **Commitment calibration** | Is the ask proportional to the relationship? (Don't ask cold visitors to "Buy Now" from an ad) |
| **Expectation setting** | Does the click trigger set clear expectations for what the landing page will deliver? |
| **Urgency authenticity** | If urgency is used, is it real? Fake urgency in the awareness/click stage poisons the entire funnel. |
| **Audience segmentation** | Are different audiences getting different messages, or is everyone seeing the same ad/email? |

### Stage 3: Landing Page Audit

| Dimension | What to Evaluate |
|-----------|-----------------|
| **Message match** | Does the page headline deliver on the click trigger's promise within 3 seconds? |
| **Promise delivery** | Does the page contain the specific information/offer that was promised? |
| **Trust escalation** | Does the page build on awareness-stage trust with deeper proof? |
| **Emotional continuity** | Does the page's tone and energy match the click trigger? |
| **Single-path focus** | Does the page funnel toward ONE action without distractions? |

### Stage 4: Conversion Action Audit

| Dimension | What to Evaluate |
|-----------|-----------------|
| **Friction inventory** | How many steps/fields/pages between CTA click and completion? |
| **Value reinforcement** | Is the value proposition restated near the point of commitment? |
| **Risk signals** | Are there security badges, guarantees, and reassurance at the moment of commitment? |
| **Cognitive load** | Does the conversion process introduce new information or decisions the visitor wasn't expecting? |
| **Recovery paths** | If something goes wrong (validation error, payment decline), is the recovery experience graceful? |

### Stage 5: Post-Conversion Audit

| Dimension | What to Evaluate |
|-----------|-----------------|
| **Confirmation clarity** | Does the visitor immediately know their action was successful? |
| **Next-step guidance** | Is it clear what happens next and when? |
| **Promise reinforcement** | Does the post-conversion experience reinforce the original promise? |
| **Buyer's remorse prevention** | Is there content that validates the decision? ("Great choice — here's what to expect") |
| **Relationship continuation** | Is there a clear path to the next stage of the relationship? (Onboarding, next purchase, referral) |

## The Funnel Continuity Scoring Rubric

Score each dimension 0-2 (0 = broken/missing, 1 = present but weak, 2 = excellent).

| # | Dimension | 0 | 1 | 2 |
|---|-----------|---|---|---|
| 1 | **Message continuity** | Core promise changes or contradicts between stages | Promise is loosely similar but language/framing shifts | Same promise threads clearly through every stage — visitor always knows "this is what I was promised" |
| 2 | **Emotional arc** | Emotional resets between stages (desire → confusion → anxiety) | Mostly consistent but 1-2 jarring transitions | Each stage builds naturally on the previous emotional state |
| 3 | **Promise escalation** | Asks for high commitment before earning trust (buy now from cold ad) | Commitment ladder exists but has gaps or premature asks | Each stage delivers value proportional to the trust it earned, then asks for slightly more |
| 4 | **Visual/tonal continuity** | Stages look and feel like different brands | Same brand but noticeable style shifts between stages | Every stage feels like the same conversation with the same entity |
| 5 | **Trust accumulation** | Trust signals appear only at one stage, or are absent entirely | Some trust signals but not strategically distributed across stages | Trust builds incrementally — each stage adds new proof appropriate to the commitment being asked |
| 6 | **ICP consistency** | Different stages target different audiences or use inconsistent language | Mostly consistent targeting but some generic stages | Every stage speaks to the same person in the same language, deepening specificity as the funnel narrows |
| 7 | **Friction management** | Unnecessary friction at multiple stages (long forms, extra pages, confusing navigation) | Some friction reduced but conversion process has avoidable steps | Every stage is as frictionless as possible — nothing between the visitor and the next step that doesn't need to be there |
| 8 | **Handoff quality** | Hard transitions between stages (new tab, lost context, login required, visual disconnect) | Transitions work but feel like separate experiences | Each stage flows seamlessly into the next — visitor never feels "lost" or "redirected" |
| 9 | **Objection resolution timing** | Objections addressed too late (after the visitor has already bounced) or not at all | Most objections addressed but some at the wrong stage | Each stage anticipates and resolves the objections appropriate to that commitment level |
| 10 | **Post-conversion follow-through** | No confirmation, no next steps, radio silence after conversion | Basic confirmation but no reinforcement, guidance, or relationship building | Clear confirmation, decision validation, next-step guidance, and a path to deeper relationship |

**Total funnel continuity score: ___ / 20**

### Score interpretation:
- **18-20: Seamless funnel.** The visitor feels like they're in one continuous conversation from first touchpoint to customer. Rare and powerful.
- **14-17: Strong funnel with gaps.** The overall journey makes sense but 1-3 handoffs are leaking conversions. Fix the weakest transitions.
- **10-13: Functional but disconnected.** The stages work individually but don't feel like a connected system. The visitor is re-orienting at each stage instead of flowing through.
- **6-9: Significant breaks.** Multiple stages contradict or undermine each other. The visitor's experience is fragmented. Major redesign of the journey needed.
- **0-5: No funnel — just stages.** Each touchpoint was designed in isolation. There's no connected journey. Start by mapping the intended path and aligning stages around a single promise.

## Funnel Leak Diagnosis

When conversion rates are low, the leak is usually at a specific handoff. Use this diagnostic framework:

| Symptom | Likely Leak Point | What to Check |
|---------|------------------|---------------|
| Low click-through rate on ads | Stage 1→2 handoff | Ad creative/copy quality, audience targeting, hook strength |
| High bounce rate on landing page | Stage 2→3 handoff | Message match between ad/email and page headline |
| High scroll depth but low CTA clicks | Stage 3 internal | CTA copy, placement, commitment level, or offer clarity |
| High CTA clicks but low form completion | Stage 3→4 handoff | Form length, field types, friction, trust signals at point of commitment |
| High cart/form abandonment | Stage 4 internal | Surprise costs, complexity, login requirements, payment issues |
| Low email open rates in nurture | Stage 2 quality | Subject lines, send timing, sender reputation, list quality |
| High refund/churn rate | Stage 5 failure | Expectation mismatch, poor onboarding, promise not delivered |
| Low repeat purchase / referral | Stage 5→1 loop | Post-conversion experience, relationship nurturing, referral incentive |

## How to Use This Skill as an Agent

When auditing a full funnel:

1. **Map the funnel stages first.** Before scoring anything, understand the complete path: what are the awareness touchpoints, what triggers the click, what's on the landing page, what's the conversion action, what happens after? Get this from the user's answers.

2. **Identify the primary funnel path.** Most businesses have multiple entry points. Focus on the HIGHEST-VOLUME or HIGHEST-VALUE path first.

3. **Crawl every touchpoint you can access.** Use search_web to crawl the landing page. If the user provides ad copy, email copy, or post-conversion details, incorporate those. If not, note the gap and work with what you have.

4. **Score message continuity first.** This is the highest-impact dimension. Extract the core promise from each accessible stage and compare them side by side.

5. **Map the emotional arc.** At each stage, identify the dominant emotion the visitor is likely feeling. Flag any resets or mismatches.

6. **Identify the leakiest handoff.** Based on the scoring and any conversion data the user provides, pinpoint where the biggest drop-off is happening.

7. **Provide stage-by-stage recommendations.** For each stage scoring 0 or 1, provide specific fixes — exact copy, exact changes, exact additions.

8. **Map fixes to downstream impact.** A fix at Stage 1 (better ad copy) cascades through the entire funnel. Quantify the downstream impact where possible.

9. **Cross-reference with LP suite outputs.** If prior scorecard, headline, ICP, layout, CTA, social proof, or mobile audit data exists in KB, incorporate those findings into the landing page stage of the funnel audit.

10. **Produce a funnel map artifact.** Visual representation of the current funnel with scores, leak points, and recommended changes at each stage.
